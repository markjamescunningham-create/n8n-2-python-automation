# linkedin-ai-content-poster

> Fully autonomous LinkedIn content machine: generates a topic, writes a post, creates an image, adds hashtags, and publishes — on a schedule.

**Category:** Marketing / AI  
**Trigger:** Schedule (every 6 hours — configurable)  
**Outputs:** LinkedIn post published with AI-generated image and hashtags

---

## What it does

On a schedule, runs a multi-agent AI pipeline:
1. **Topic Generator** — creates high-value content ideas using a strategic prompt (AI/automation/LinkedIn growth themes)
2. **Content Creator** — expands the best topic into a full LinkedIn post + image description
3. **Image Generator** — creates a realistic LinkedIn-appropriate image via OpenAI DALL-E
4. **Hashtag Generator** — produces broad, niche, and trending hashtags via a separate AI agent
5. **Publisher** — merges everything and posts to LinkedIn as an organisation page

Great for: personal brand automation, agency social media management, thought leadership on autopilot.

## Setup

### n8n
1. Import `workflow.json` into n8n
2. Add **OpenAI** credential (needs both Chat and Image API access)
3. Add **LinkedIn OAuth2** credential (requires a LinkedIn app with `w_member_social` or `w_organization_social` scope)
4. Update the content topic generator prompt to match your brand voice and topics
5. Set your posting frequency in the Schedule Trigger node
6. Activate

### Python
1. `pip install -r requirements.txt`
2. Copy `.env.example` → `.env`
3. Generate and post once: `python workflow.py`
4. Schedule with cron for automated posting

## Environment Variables

| Variable | Description |
|----------|-------------|
| `OPENAI_API_KEY` | Your OpenAI API key |
| `OPENAI_MODEL` | Chat model (default: `gpt-4o-mini`) |
| `OPENAI_IMAGE_MODEL` | Image model (default: `dall-e-3`) |
| `LINKEDIN_ACCESS_TOKEN` | LinkedIn API access token |
| `LINKEDIN_AUTHOR_URN` | Your LinkedIn profile URN (e.g. `urn:li:organization:12345`) |
| `OUTPUT_DIR` | Directory to save generated images locally (default: `./output`) |

## Notes

- The **topic generator** is pre-configured with an AI/automation/LinkedIn growth brief — update the system prompt in the node to match your niche
- LinkedIn's API requires a **verified app** and may have posting rate limits
- Images are generated by DALL-E and uploaded via LinkedIn's asset API before posting
- The Python version saves the generated post and image locally before posting, giving you a review step if needed

## Extending

- Add a **"human review" step**: instead of posting directly, save to Notion or Airtable for approval first
- Add **multiple brand voices** using a random selector node
- Connect to a **content calendar** in Google Sheets to schedule specific topics in advance
